GitHub Repository:
https://github.com/akhilkumrawat/Assignments-Portfolio_DLMCSPSE01

How to Run Locally:
1. Clone the repository:
   git clone https://github.com/akhilkumrawat/Assignments-Portfolio_DLMCSPSE01.git

2. Install frontend dependencies:
   cd frontend
   npm install
   npm start
   (Runs on http://localhost:3000)

3. Install backend dependencies:
   cd backend
   npm install
   Create a .env file with the following content:
     MONGODB_URI=mongodb://localhost:27017/travelworld
     PORT=5000

4. Start MongoDB service (ensure mongod is running)
   mongod

5. Run backend:
   node server.js
   (API runs on http://localhost:5000)

Login Credentials:
No login is required for this application.

Optional:
- Use MongoDB Compass to view saved contact form entries.
